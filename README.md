# dionisisdiamantis.com

Personal portfolio website for Dionisis Diamantis.

## Tech

- HTML
- CSS
- Vanilla JavaScript

## Run locally

No build tools are required.

### Option 1: Python

```bash
python -m http.server 5500
```

Then open [http://localhost:5500/portfolio.html](http://localhost:5500/portfolio.html).

### Option 2: VS Code Live Server

Open `portfolio.html` and use **Open with Live Server**.

## Project structure

- `portfolio.html` - Main HTML page
- `css/styles.css` - Stylesheet
- `js/script.js` - JavaScript logic
- `assets/images/me.jpeg` - Profile image

## Deployment

This project can be deployed as a static site using:

- GitHub Pages
- Netlify
- Vercel
